# AI 广告图片生成器

一个功能强大的 Next.js 应用，帮助您快速生成适用于各大平台的专业广告图片。

## 🚀 功能特性

### 📸 图片上传
- 支持 JPG, PNG 格式图片上传
- 拖拽上传或点击上传
- 实时预览上传的产品图片
- **多图片支持**：上传最多5张产品图片，批量生成广告
- **图片比例设置**：自定义图片在画布中的宽度和高度比例
- **自定义宽高比**：支持多种预设宽高比（1:1、4:3、16:9等）

### ✏️ 文字编辑
- 添加多个广告文字组
- 支持顶部、底部和自定义位置文字
- 自定义文字颜色（支持颜色选择器）
- 动态添加和删除文字
- **文字大小控制**：通过滑块调整文字大小比例
- **多字体支持**：选择不同字体（Arial、Times New Roman、微软雅黑等）
- **文字组选项**：每个文字组支持多个文案选项，批量生成不同组合
- **精确定位**：支持通过拖拽或输入百分比精确定位文字位置
- **全局拖拽支持**：所有文字组均支持在预览中直接拖拽调整位置

### 🎨 CTA 按钮定制
- 自定义按钮背景颜色
- 自定义按钮文字颜色
- 多种圆角样式选择（无圆角、小圆角、中等圆角、大圆角、胶囊形）
- **按钮大小控制**：通过滑块调整按钮大小比例
- **按钮位置控制**：通过拖拽或滑块精确定位按钮位置
- **多按钮文案**：支持添加多个CTA按钮文案选项，批量生成不同组合
- 实时预览按钮效果
- **按钮模板库**：提供20种预设按钮样式模板，一键应用
- **组合模板**：提供20种文字+按钮组合模板，快速创建完整广告设计

**✨ 平台选择**
- 支持选择生成哪些平台的广告图片
- 默认全选所有平台
- 可按平台分类选择（Facebook、Google Ads、Instagram等）
- 实时显示选择的平台数量和总图片数
- **自定义尺寸**：支持为每个平台自定义图片尺寸（100-3000像素）
- **美观界面**：采用卡片式设计，支持内联尺寸编辑
- **多平台预览**：支持选择查看不同平台的预览效果，实时显示尺寸信息

### 🌐 多语言支持
- 支持中文和英文界面
- 一键切换语言
- 根据语言自动调整下载文件名格式
- **自动语言检测**：根据浏览器语言自动选择对应界面语言
- **智能语言回退**：非中文浏览器环境默认使用英文界面
- **内容国际化**：所有模板和组件内容根据当前语言智能切换显示对应语言内容

### 🎭 模板系统
- **按钮模板**：20种精心设计的CTA按钮样式，一键应用
- **组合模板**：20种完整的广告设计方案，包含文字和按钮样式
- **智能语言适配**：模板内容根据当前界面语言自动切换中英文
- **模板预览**：实时显示当前选择的模板效果
- **自定义调整**：应用模板后可进一步微调各项参数

### 🖥️ 用户体验优化
- **滚动跟随预览**：右侧预览面板使用sticky定位，在用户滚动页面时始终保持在视窗右侧
- **响应式布局**：大屏幕使用三列布局，小屏幕自动调整为单列布局
- **美观滚动条**：自定义滚动条样式，提升视觉体验
- **视觉指示器**：预览面板标题旁显示"📌 跟随滚动"提示
- **大尺寸预览**：预览图最大高度700px，确保用户能清楚看到细节
- **拖拽交互**：支持直接在预览中拖拽调整文字和按钮位置
- **精确定位**：拖拽位置自动四舍五入到整数像素，确保清晰显示

### 📦 批量下载
- 一键生成所有平台的广告图片
- ZIP 格式打包下载
- 高质量 PNG 输出
- **智能文件命名**：根据所选语言自动使用对应语言的文件名
- **多图片组合**：自动生成所有文字组合和按钮文案组合的广告图片

## 📖 使用指南

### 1. 上传产品图片
- 点击上传区域或拖拽图片到指定区域
- 支持 JPG、PNG 格式，最多上传5张图片
- 使用图片比例设置调整图片在画布中的显示大小

### 2. 添加广告文字
- 点击"添加文字组"按钮
- 输入广告文案，可添加多个文案选项
- 选择文字位置（顶部/底部/自定义）
- 自定义文字颜色和字体
- 调整文字大小比例
- 对于自定义位置的文字，可通过滑块或直接在预览中拖拽调整位置

### 3. 设计 CTA 按钮
- 设置按钮背景颜色和文字颜色
- 选择按钮圆角样式和字体
- 添加多个按钮文案选项
- 调整按钮大小比例
- 通过滑块或直接在预览中拖拽调整按钮位置
- **使用模板**：从下拉菜单选择预设按钮模板，快速应用样式
- **使用组合模板**：从下拉菜单选择完整广告设计方案，一键应用文字和按钮样式

### 4. 调整图片比例
- 选择自动模式（保持原始比例）或自定义模式
- 调整图片在画布中的宽度和高度比例
- 在自定义模式下可选择预设宽高比（1:1、4:3、16:9等）

### 5. 选择生成平台
- 在"平台选择"区域选择要生成的平台
- 支持全选/取消全选功能
- 可按平台分类选择（Facebook、Google Ads、Instagram等）
- 实时显示选择的平台数量和总图片数
- **自定义尺寸**：点击✏️按钮可编辑每个平台的图片尺寸
- **重置尺寸**：点击↺按钮可重置为平台默认尺寸

### 6. 预览和调整
- **多平台预览**：在右侧预览区域选择要查看的平台
- **实时预览**：切换平台时自动更新预览效果
- **尺寸信息**：预览图左下角显示当前平台的尺寸
- **拖拽调整**：在预览中直接拖拽调整文字和按钮位置
- **刷新预览**：点击刷新按钮重新生成预览

### 7. 生成并下载
- 点击"生成广告图片"按钮
- 系统会自动生成所选平台、所有文案组合的广告图片
- 下载 ZIP 文件包含所有生成的广告图片

## LeanCloud 配置指南

### 1. LeanCloud 账户设置

1. 访问 [LeanCloud 官网](https://www.leancloud.cn/) 并注册账号
2. 创建新应用，选择开发版（免费）
3. 进入应用 > 设置 > 应用凭证，获取以下信息：
   - AppID
   - AppKey
   - 服务器地址(serverURL)

### 2. 配置应用

1. 打开文件 `src/services/feedbackService.ts`
2. 替换以下配置项：
   ```typescript
   // 替换为你从LeanCloud获取的实际值
   const LEANCLOUD_APP_ID = '你的AppID';
   const LEANCLOUD_APP_KEY = '你的AppKey';
   const LEANCLOUD_SERVER_URL = '你的服务器地址';
   ```

### 3. 创建数据结构

1. 在LeanCloud控制台中，进入 "存储" > "数据"
2. 创建新的Class，命名为 "Feedback"
3. 权限设置为 "无限制"

### 4. 查看收集的数据

1. 进入LeanCloud控制台 > 应用 > 存储 > 数据 > Feedback
2. 在此可以看到所有用户提交的反馈
3. 可以点击 "导出" 按钮将数据导出为CSV格式

### 功能特点

- 自动检测用户地区，在中国大陆自动使用LeanCloud，其他地区使用Google Sheets
- 即使用户离线，反馈也会保存在本地存储中
- 支持跟踪用户UUID，便于分析用户行为
